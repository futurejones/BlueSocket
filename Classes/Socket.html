    Declaration

    public func connect(to host: String, port: Int32, timeout: UInt = 0, familyOnly: Bool = false) throws

    Parameters

    host

    The host name to connect to.
    port

    The port to be used.
    timeout

    Timeout to use (in msec). Note: If the socket is in blocking mode it will be changed to non-blocking mode temporarily if a timeout greater than zero (0) is provided. The returned socket will be set back to its original setting (blocking or non-blocking).
    familyOnly

    Setting this to true will only connect to a socket of the family of the current instance of Socket. Setting it to false, will allow connection to foreign sockets of a different family. Default is false.

    public func listen(on port: Int, maxBacklogSize: Int = Socket.SOCKET_DEFAULT_MAX_BACKLOG, allowPortReuse: Bool = true, node: String? = nil) throws

    Parameters

    The port to listen on.
    maxBacklogSize
